# NHS App Template Repository
This repository is used as a template for any _new_ repositories made by the NHS App team.

It has all the necessary setup configured to follow the convention and standards created during the [GitHub Migration Project](https://nhsd-confluence.digital.nhs.uk/x/XqOqNg).
