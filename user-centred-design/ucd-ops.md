# UCD ops

The NHS App has an Ops team for user-centred design. 

This team includes: 

- Profession leads for user research, design, and content (Simon Davis, Mike Gallagher, and Auriol Palladino, respectively)
- Cluster leads (i.e. cluster managers for all things UCD; Tanja Galloway and Flora Macleod)
- User research administration support
- Content and design system support

If you are confused about how something works on the App and it is related to user-centred design, these people are probably where you should start.

You can @ the team on Slack by using `@nhsapp-ucdops`.
